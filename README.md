 🚀 Project 1: Stock Intelligence Valuation Tool
File:`Stock_Intelligence_Valuation_Tool.ipynb`

 💡 The User Problem
Retail investors often struggle to interpret raw financial data and technical charts, leading to emotional rather than data-driven decisions.

🤖 The AI Solution
I built an end-to-end decision support system that:
- Aggregates real-time financial data using YFinance.
- Uses Scikit-Learn to predict market trends.
- contextualizes P&L data to offer explainable "Buy/Hold" signals.

🛠 Tech Stack
- Python & Pandas for data engineering.
- Machine Learning (Sklearn)for predictive modeling.
- Plotly/Gradio for the visualization dashboard.

🎓 Project 2: AI Career Decision Platform
File: `AI_Career_Decision_Platform.ipynb`

💡 The User Problem
Students are overwhelmed by career choices and generic advice that doesn't account for their specific aptitude or market trends.

🤖 The AI Solution
A personalized recommendation engine using 8,000+ synthetic personas. It uses a hybrid rule-based + ML pipeline to match user profiles with optimal career paths.

🛠 Tech Stack
- TensorFlow for pattern matching.
- Synthetic Data Generation for model training.
